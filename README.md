# phase-ai-anime-skill

A TypeScript-based AI skill for anime-related queries on the Phase platform.

## Requirements

- Node.js >= 16
- npm >= 8

## Getting Started

### Install dependencies

```bash
npm install
```

### Build

```bash
npm run build
```

### Test

```bash
npm test
```

### Test with coverage

```bash
npm run test:coverage
```

### Lint

```bash
npm run lint
```

### Lint and auto-fix

```bash
npm run lint:fix
```

## Project Structure

```
phase-ai-anime-skill/
```
