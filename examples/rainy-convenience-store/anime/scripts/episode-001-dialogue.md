# Episode 001 Dialogue Script

## Title

雨夜耳机失踪案

## Logline

怕水的猫耳侦探在暴雨夜追查一只会说话的失踪耳机，却发现耳机掌握的案情指向她自己。

## Beat Sheet

| Beat | Time | Function | Story Action |
| --- | ---: | --- | --- |
| beat-001 | 0-3s | Hook | 冰柜倒影里传来失踪耳机的耳语，侦探被迫接案。 |
| beat-002 | 3-12s | Choice | 她必须离开安全的干燥门垫，向雨声最重的通道前进。 |
| beat-003 | 12-24s | Clue | 冷柜旁出现湿脚印和异常波形，耳机提示“水会冲走记忆”。 |
| beat-004 | 24-36s | Reveal | 她找到耳机，但耳机称自己不是失踪物，而是证据。 |
| beat-005 | 36-49s | Action | 漏水漫过地砖，她克服怕水，用围巾卷回耳机。 |
| beat-006 | 49-60s | Twist | 耳机播放她自己的求救录音，暗示真正失踪的是侦探本人。 |

## Dialogue And Subtitle Text

| Cue | Speaker | Text | Performance |
| --- | --- | --- | --- |
| cue-002-dialogue-whisper | 耳机 | 你终于听见我了。 | 贴耳、带电流噪声。 |
| cue-005-dialogue-detective | 猫耳侦探 | 我讨厌下雨，但更讨厌谜题逃走。 | 紧张，尾音强撑镇定。 |
| cue-009-dialogue-earbud-clue | 耳机 | 别踩水，水会把记忆冲走。 | 平静，像在背诵案卷。 |
| cue-010-dialogue-detective-deduction | 猫耳侦探 | 湿脚印没有脚跟，犯人不是人。 | 快速推理，小声确认。 |
| cue-013-dialogue-earbud-reveal | 耳机 | 我不是失踪物，我是证据。 | 清晰、近距离。 |
| cue-014-dialogue-detective-snap | 猫耳侦探 | 会说话还会诬告，嫌疑更大。 | 逞强，带一点可爱怒气。 |
| cue-017-dialogue-detective-brave | 猫耳侦探 | 怕水不是证据，逃走才是。 | 深呼吸后坚定。 |
| cue-021-dialogue-twist | 耳机 | 别找耳机了。真正失踪的人，是你。 | 几乎无情绪，最后两个字压低。 |

## Voice Direction

- 猫耳侦探：年轻女性声线，节奏快，紧张时会用推理术语保护自己。
- 耳机：中性电子耳语，音量小但贴近，偶尔出现短促 glitch。
- 字幕：每句不超过两行，保留悬疑信息，不解释世界观。
